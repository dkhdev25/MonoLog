# MonoLog  
A minimalist website for writing personal devlogs. Designed for developers who value an organized, simple, and productive environment for documenting their creative process.  

## About
A devlog editor built for the One-Key Challenge on Hacklcub, by using a single-key system for navigating, writing, and managing your entries.

Free to use!  
https://dkhdev25.github.io/MonoLog/  

## Usage
Instructions in the project.

Controls: one key to navigate and type, mouse for selecting where to edit.

## Features
- Only one key and a mouse needed.
- Letters, symbols, and numbers can be typed.
- Simple and organized interface.
